Structs

Channel

The internal implementation of a channel representation backed by a VecDeque<Message>. Readers and writers to this channel must increment the reader/writer count. This is implemented for ChannelWriter/ChannelReader, which will increment/decrement readers/writers when cloning/dropping.

ChannelReader

Reader end to a Channel. ChannelReader implements Clone and Drop to automatically update the underlying channel.

ChannelWriter

Writer end to a Channel. ChannelWriter implements Clone and Drop to automatically update the underlying channel.
